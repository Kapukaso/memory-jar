# Memory-Jar

A simple web app named **Memory-Jar**, built with JavaScript, that allows users to capture, store, and revisit their memorable moments in a digital “jar”.

## Features
- Add, edit and delete memory entries.
- View all saved memories in a clean UI.
- Client-side persistence through localStorage.
- Minimal and fast JavaScript frontend.

## Getting Started

### Prerequisites
- Node.js
- npm or yarn

### Installation
```bash
git clone https://github.com/Kapukaso/memory-jar.git
cd memory-jar
npm install
````

### Running Locally

```bash
npm start
```

Open [http://localhost:3000](http://localhost:3000) to view the app.

To create a production build:

```bash
npm run build
```

## Usage

* Open the web app in your browser.
* Add a new memory with a date and description.
* Edit or delete existing memories.
* View all stored memories using the built-in interface.
